# mcfeed

Simple, scrollable feed-like interface that displays articles from raw JSON data.

## Installation

1. Clone project
2. Open Terminal and navigate to the directory that contains the project.
2. Run: pod install
3. Open MCFeed.xcworkspace

## Requirements

Xcode 8.2.1+
iOS 10.0+
